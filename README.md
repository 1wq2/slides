# Slides App
***
### What is this app for?

This application let you qyuckly create an lightweight presentations and deploy it on production url to share with your friends or co-workers.

### Requirements

Modern browser with JavaScript enabled.
[Check it out.](https://www.whatismybrowser.com/) [And update, if necessary.](https://browsehappy.com/)
